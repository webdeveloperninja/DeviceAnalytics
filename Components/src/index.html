All Things Sensors - Analytics Web Component Library

We are a Web Component first company
Usage

Add the script to your website. Choose either ES2015 version or ES5 depending on your browser support needs.

https://device-analytics.allthingssensors.com/main-es5.js

https://device-analytics.allthingssensors.com/main-es2015.js

Finally add the component selector with the appropriate inputs and outputs defined in the component documentation.

Device Events Component

Add the following element to your website. device-events-grid with the device-id attribute as the device id.

Device Line Graph Component

Add the following element to your website. device-events-line-graph with the device-id attribute as the device id.
